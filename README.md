# Minha-Gravacoes-Audio
Apresento neste vídeo um dashboard desenvolvido no Excel, utilizando funções essenciais como SOMA, MÉDIA, ARRED, MÍNIMO, MÁXIMO, MAIOR, MENOR, SUBTOTAL e TRANCAMENTO TOTAL. Este projeto foi pensado para trazer uma visão clara, organizada e interativa dos dados, com foco em eficiência e análise estratégica.
